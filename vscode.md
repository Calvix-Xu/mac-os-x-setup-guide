# Visual Studio Code

## 1. 配置从命令提示行启动`vscode`

添加如下命令到 `env.sh` 里。

```
alias vscode='open -a "Visual Studio Code" $1'
```

注：`env.sh` 的具体配置参见 `iTerm2` 章节。

## \*2. 安装 vim 插件

按 `command + shift + p` 然后在输入框中输入「扩展」。

![](/assets/Vscode - Extension.png)

在应用商店里搜索名为`Vim`的扩展，安装即可。

![](/assets/install-vim.png)
