# Mac-OS-X-Setup-Guide

本书主要是关于 Mac OS X 的设置，包括了常用的各种软件和编程语言的配置。

本书的配置大部分适用于科研工作者/程序员/码农，也适用于有志于从事类似工作的学生，对于Mac OS 的日常使用的帮助不大。

主要原则是简洁，并尽量遵循最佳实践。
